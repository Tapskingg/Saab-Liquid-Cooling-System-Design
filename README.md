# Saab Liquid Cooling System Design for Aerospace Electronics

Development of an additively manufactured liquid cooling system for future aerospace electronics through CAD design, CFD simulation, design optimisation, and experimental validation.

---

## Project Overview

This project was completed during my Mechanical Engineering internship at Saab AB.

The objective was to design, analyse and optimise an additively manufactured liquid cooling system capable of dissipating **500 W** while maintaining low coolant temperature rise, uniform flow distribution and manufacturability.

The work combined computational analysis with physical testing to validate design performance.

---

## Engineering Objectives

- Dissipate 500 W thermal load
- Minimise coolant temperature rise
- Reduce pressure losses
- Improve coolant distribution
- Minimise hotspots
- Design for powder-bed additive manufacturing
- Validate CFD predictions experimentally
